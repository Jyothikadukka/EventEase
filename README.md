# 🎉 EventEase - Event Management Web App

EventEase is a modern web application built using Blazor (.NET 8) that allows users to browse events, view details, and register for participation. The application demonstrates component-based development, routing, state management, and performance optimization.

---

## 🚀 Features

- 📌 Event Card Component with dynamic data binding  
- 🔗 Smooth Navigation using Blazor routing  
- 📝 Registration Form with validation  
- 🔐 User Session Management using services  
- ✔️ Attendance Tracker for events  
- ⚡ Performance optimization using efficient rendering  

---

## 🛠️ Technologies Used

- Blazor (.NET 8)
- C#
- HTML & CSS
- Git & GitHub

---

## 🤖 Role of Microsoft Copilot

Microsoft Copilot assisted throughout development by:

- Generating reusable components (Event Card)  
- Suggesting routing structures and navigation  
- Helping debug errors and improve validation  
- Assisting in implementing state management  
- Recommending performance optimizations
